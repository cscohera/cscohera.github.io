Knight Runners Weather Alert Bot

Made this Bot to tell UCF club runners the weather. This was done so that if I forgot to check Thorguard (the weather system used by UCF) club members would still know if practice was happening or not. The Bot started off very simple with me learning the basics I wanted to get my data from UCF's weather system Thorguard as that determines if we have practice or not. After doing some research I found I need to webscape as they did not have a API for me to pull from. Once I was able to figure out what I wanted to display I started to perfect it. This is when I ran into a larger problem when I had a few other officers test the bot. I couldn't pull information fast enough and multiple requests would mess up the formatting of the bot. Additonally, I hit a rate limit from discord which were to major things I needed to fix. In order to fix both of these issues I decided to start caching weather info for 30 seconds so that if someone asks for the weather it will take the first request and requests after which will be ignored inorder to not send out unneed requests. Final product: Bot can display weather at any time with the !weather command and I have also created a loop command that will display the weather 45 mins before practice everyday.
